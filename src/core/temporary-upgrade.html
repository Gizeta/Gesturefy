Gesturefy Major Update 2.0

Gesturefy has been Upgraded to version 2.0. Please take a few minutes to read about the changes.

This update adds a lot of different requested features. However there are two major changes that were worked out in this release that everyone should know about.

The first change is that your list of gestures is no longer static. This means that you can now create gestures and assign any command to them. While this may seem unnecessary let me point out some use cases and benefits:

  • You can now use multiple different gestures to perform different web searches or open different websites. In the older version you were only able to set exactly one search engine or one specific website to one specific gesture.
  • Every command with advanced settings has now its own settings, where before one setting sometimes applied to multiple commands. Due to this some commands like "Reload" & "Reload Cache" are now pooled into one command.
  • It also enables the support for "Send Message To Other Add-on" (this allows you to execute other add-ons functionality if the addon supports it) as well as "Custom User Scripts" (this is not included in the current update because we are waiting for an API which is supposed to land in Firefox 63)
  • Your list of gestures in the addon settings is now much more clearer, since it only shows the gestures you are currently using.

The second change is the amount of permissions Gesturefy requires on install. Since a lot of people complained about it (which is reasonable in some way) we minimized the amount of fixed permissions. You will now be prompted to allow further permission if you use a command which requires additional permissions. Note that your defined wheel and rocker commands including some advanced command settings couldn't be transitioned to the new config, wherefore you have to reconfigure them manually (sorry for that).

As a side effect not all of your old Gestures could automatically be converted to the new config format, you can however click the button below, which will restore them once you accepted the required permissions.

Additional noteworthy changes:

  • Settings are now synced accross your devices (if you have enabled it in Firefox)
  • You can now disable Gesturefy for certain websites via blacklist.
  • Setting UI overhaul including a dark and a white theme.
  • More commands, command settings and fixes. For more information read the full changelog.

We have put a lot of time and effort into this, so any feedback about the new version is appreciated and can be submitted here.

Enjoy browsing the web :-)
